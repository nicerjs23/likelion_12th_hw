Django MTV 패턴이란?

장고는 MTV(Model,Template,View)패턴을 기반으로하는 프레임워크이다.

Model은 데이터 모델을 의미
Template는 사용자에게 보여지는 부분으로 인터페이스이다.
View는 요청에따른 적절한 로직을 수행한다. 함수라고 생각하면 이해하기 편하다.

사용자가 url을 통해 보낸 request가 다시 reponse되는 과정

  1. 장고 서버로 사용자가 url을 통해 request가 들어옴

  2. URL Conf를 통해 url을 분석하여 적당한 view를 매핑함

  3. view는 요청에 따른 적절한 로직을 수행하여 model에 전달

  4. model은 db와 연결하여 view로 필요한 데이터를 보내줌

  5. template에서 인터페이스를 만듬

  6. 사용자에게 response
